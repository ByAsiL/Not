# İhale Hafızası PWA

Bu paket, mobilde uygulama gibi çalışacak PWA ön izlemesidir.

## Telefonda uygulama gibi kullanma

1. Dosyaları GitHub reposuna yükle.
2. Repository > Settings > Pages bölümünden GitHub Pages'i aç.
3. Verilen linki Android telefonda aç.
4. Chrome menüsünden "Ana ekrana ekle" veya "Uygulamayı yükle" seç.
5. Ana ekrandan açıldığında adres çubuğu olmadan uygulama gibi çalışır.

## İçerik

- İhale Kaydet
- Kayıtlı İhaleler
- Not Kaydet
- Şifre Kasası
- Gelir Gider
- Yapılacaklar
- Koyu Android tema
- LocalStorage kayıt
- PWA manifest
- Service worker

## Önemli

Şifre kasası bu ön izlemede düz localStorage kullanır. Gerçek kullanımda ana parola ve şifreleme eklenmelidir.
